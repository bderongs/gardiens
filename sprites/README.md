# Sprites Universal LPC

Ce dossier contient les sprites du Universal LPC Spritesheet.

## Structure

- `body/` - Parties du corps (tête, torse, jambes, bras)
- `hair/` - Différents styles de cheveux
- `clothes/` - Vêtements (hauts, bas, robes)
- `accessories/` - Accessoires (chapeaux, etc.)

## Comment obtenir les sprites

1. **Via le générateur en ligne** (recommandé pour commencer) :
   - Aller sur : https://liberatedpixelcup.github.io/Universal-LPC-Spritesheet-Character-Generator/
   - Créer un personnage avec les options souhaitées
   - Exporter le sprite final
   - Le placer dans le dossier approprié

2. **Via GitHub** (pour les sprites individuels) :
   - Dépôt : https://github.com/makrohn/Universal-LPC-spritesheet
   - Télécharger le repository
   - Extraire les sprites nécessaires dans les dossiers correspondants

## Format des fichiers

Les sprites doivent être en PNG avec transparence.
Nommage recommandé : `[type]_[variation].png`
Exemples :
- `hair_short_blonde.png`
- `clothes_shirt_blue.png`
- `body_head_light.png`
