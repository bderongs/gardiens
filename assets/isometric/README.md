# Isometric Prototypes Tiles - Kenney Assets

## Instructions de téléchargement

1. Téléchargez le pack depuis : https://kenney-assets.itch.io/isometric-prototypes-tiles
2. Extrayez les fichiers dans ce dossier
3. Organisez les tuiles comme suit :
   - `tiles/` : Toutes les tuiles isométriques
   - `characters/` : Sprites de personnages (si inclus)

## Structure attendue

```
assets/isometric/
├── tiles/
│   ├── floor_*.png
│   ├── wall_*.png
│   ├── door_*.png
│   └── objects_*.png
└── README.md
```

## Spécifications techniques

- Taille des tuiles : 256x512 pixels
- Largeur de tuile : 256px
- Hauteur de sol : 128px
- Angle de vue : 30° × 45°
- Licence : CC0 1.0 Universal
